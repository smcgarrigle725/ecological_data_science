---

## Notes

- All figures produced on the full dataset (no train/test split — this is observational field data, not a predictive model)
- Continuous predictors are not standardised for plotting; raw concentrations and isotope values are used
- Bray-Curtis dissimilarity on stable isotopes will trigger a warning about negative values (δ¹³C); this is expected and documented in the manuscript
